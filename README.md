# Smart Academy Tailwind Cart App

A simple cart interface built with Next.js, React, and Tailwind CSS. It fetches cart data from an external API and displays products with pricing and totals.

## Tech Stack

* Next.js 16
* React 19
* Tailwind CSS 4

## Features

* Fetches cart data from https://dummyjson.com/carts/1
* Displays product list with image, title, quantity, and price
* Shows total items, quantity, and discounted total
* Responsive layout using Tailwind CSS

## Installation

```bash
git clone https://github.com/your-username/smart-academy-tailwind-four.git
cd smart-academy-tailwind-four
npm install
```

## Development

```bash
npm run dev
```

Open http://localhost:3000

## Build and Start

```bash
npm run build
npm start
```
